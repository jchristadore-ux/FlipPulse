Create a Kalshi account

Kalshi is the regulated exchange where your trades happen.

  1. Open the Create Your Kalshi Account button from the start page or your welcome email.
  2. Enter your email and choose a password.
  3. Click Create account.
ℹ️ Already have Kalshi?

That's fine — just log in and skip to Section 2.

Verify your email

  1. Find the verification email from Kalshi in your inbox and click the link inside it.
✓

What you should see: a "verified" confirmation, then you're returned to Kalshi, logged in.

Identity verification (if asked)

To trade real money, Kalshi may ask you to confirm your identity (name, date of birth, last 4 of your SSN). Follow the prompts — this is Kalshi's standard process for any US exchange, not ours.

ℹ️ You can keep going

You can finish this whole guide even while identity check is "pending." It only needs to be approved before you fund and go live — a later step.

Log in

✓

What you should see: the Kalshi dashboard with your balance (likely $0.00 — funding comes later).

2

Generate Kalshi API credentials

The most important section. You'll collect Thing #1 (Key ID) and Thing #2 (PEM file) here.

Open API settings

  1. Click Settings (or your profile menu), then find API Keys.
  2. Click Create API Key.
💡 Can't find it?

It may live under Settings → Security → API or a "Developers"/"API Access" menu. Look for the word API. On a phone, use the Kalshi website in a browser — API keys are far easier there.

Name and create the key

  1. Give the key a name — we suggest FlipPulse.
  2. Click Create key.

Copy the Key ID and download the PEM

  1. Copy the API Key ID — a long code like a1b2c3d4-0000-4e5f-9a8b-1234567890ab. Paste it into your note. (This is Thing #1.)
  2. Download the PEM file. A file like kalshi-flippulse.pem saves to your Downloads folder. (This is Thing #2.)
🔴 CRITICAL — the PEM downloads only once

Kalshi shows the private key a single time. If you leave this page without downloading, you can't get the same one back — you'd have to make a new key. Download it now.

Save the PEM safely

Move the file somewhere you'll find it (e.g. a FlipPulse folder). You'll simply upload the file as-is at the end. Don't rename its contents, open it in Word/Notes and re-save, or copy the text out — that can corrupt it.

✓

What you should see: the Key ID in your note, and a .pem file in your Downloads folder.

If something goes wrong

ProblemFix
Can't find API settingsUse the Kalshi website in a browser; look under Settings → Security; search the page for "API".
PEM won't downloadTry another browser (Chrome, Edge, Safari, Firefox); turn off pop-up/download blocking for kalshi.com.

4

Create your Telegram bot

You'll make your personal alert bot by chatting with Telegram's official bot-maker, BotFather.

  1. Tap the search bar and type BotFather.
  2. Tap BotFather — the result with the blue verified check (✓). There are fakes; the checkmark is how you spot the real one.
  1. Tap Start (or the START button at the bottom).
  2. In the message box at the bottom, type /newbot and send it.
💡 Where do I type?

The message box is the long bar at the very bottom of the chat — the same place you'd type "hello" to a friend. Type /newbot and press the send arrow (➤).

5

Name your bot

BotFather asks two quick questions.

  1. Bot name — a friendly label, e.g. FlipPulse Alerts. Send it.
  2. Bot username — a unique @handle that must end in bot and has no spaces.

✅ Valid usernames

  • flippulse_jane_bot
  • janes_flippulse_bot
  • jane2026alertsbot

❌ Invalid usernames

  • flippulse_jane — no "bot"
  • FlipPulse Jane bot — spaces
  • flippulse_bot — taken
💡 "Username already taken"?

Normal — just add numbers or your name until it's accepted, e.g. flippulse_jane_8391_bot.

6

Copy the bot token

This is Thing #3. Treat it like a password.

  1. Tap the token once to copy it (on desktop, select the line and press Ctrl+C / ⌘+C).
  2. Paste it into your note. (This is Thing #3.)
🔴 Never share your bot token

Anyone with it can control your alert bot. We only ask for it inside the secure form. FlipPulse staff will never ask for it by email, text, or phone.

✓

What you should see: the full token in your note — a number, a colon :, then a long string of letters. Make sure you copied everything after the colon too.

7

Open your bot

Open the bot you just made — this is your bot, not BotFather.

  1. In BotFather's message there's a link like t.me/flippulse_jane_bot — tap it. Or search your bot's username to find it.
  2. You'll land in a fresh, empty chat with your bot's name at the top.
💡 Right chat?

The name at the top should be your bot's name (e.g. "FlipPulse Alerts"), not "BotFather".

8

Click Start

One tap that's easy to miss — and essential.

  1. Tap START at the bottom of your bot's chat.
🔴 This step is required

Telegram blocks bots from messaging you until you press Start first. Skip it and your bot simply can't send alerts — even though everything else is right.

✓

What you should see: the START button disappears and a normal message box appears. That's the sign it worked.

9

Get your Telegram Chat ID

The last item — Thing #4 — from another official helper, @userinfobot.

  1. Tap the search bar and type userinfobot.
  2. Open @userinfobot and tap Start.
  3. Send the message hi (any message works).
  4. It replies with a line Id: 123456789. Copy only the number. (This is Thing #4.)
✓

What you should see: a reply containing Id: and a number (usually 8–10 digits). Copy just the digits.

💡 Numbers only

The Chat ID is only digits (sometimes a leading minus for groups). If you copied any letters, grab just the number after Id:.

FAQ & troubleshooting

Almost every question people ask, answered.

I lost my PEM file.
The PEM can't be re-downloaded, but a new key takes 60 seconds. Go to Kalshi → Settings → API Keys → Create API Key, download the new PEM, copy the new Key ID, and use that pair. You can delete the old key.
I can't find the API settings on Kalshi.
Use the Kalshi website in a normal browser (not the phone app). Look under Settings, then Security or API, and search the page for "API".
My bot token isn't working / the bot never messages me.
Two usual causes: (1) you didn't press START in your bot's chat (Section 8) — open your bot and tap Start; or (2) the token was copied incompletely — re-copy the entire token, including everything after the colon. You can get it again from BotFather with /mybots → your bot → API Token.
I don't see the Start button.
If the chat already shows a normal message box, it's already started — you're fine. If unsure, type /start in the message box and send it.
@BotFather isn't responding.
Make sure you opened the verified BotFather (blue ✓). Send /start again. Still silent? Reopen Telegram, check your internet, and retry. Avoid look-alike bots without the checkmark.
@userinfobot isn't responding.
Confirm you opened @userinfobot (exact spelling), tapped Start, and sent a message like hi. If nothing comes back, reopen Telegram and retry. A backup helper is @getidsbot.
I accidentally created two bots.
Harmless. Decide which to use and copy that bot's token (BotFather → /mybots → pick the bot → API Token). Delete the extra with /deletebot if you like.
I downloaded the wrong Telegram app.
Delete it and reinstall the official Telegram (blue paper-plane, by "Telegram FZ-LLC") using the links in Section 3.
My Chat ID isn't appearing.
Make sure you sent a message to @userinfobot (like hi) after tapping Start — it only replies once you message it. The ID is the number on the Id: line; copy just the digits.
Is my money safe? What can the API key do?
The key only lets your bot place trades on your own Kalshi account. It cannot withdraw funds. Your money stays in your Kalshi account the whole time.
Do I need to fund my Kalshi account now?
No. You can finish setup and watch the bot in paper mode (fake money) first. Funding and going live are separate steps you choose later.
