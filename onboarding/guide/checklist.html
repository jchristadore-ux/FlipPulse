🎯 The 4 things you're collecting — write them here

1 · Kalshi Key ID
2 · Kalshi PEM file ☐ Saved in my Downloads folder (upload the file — don't retype it)
3 · Telegram Token
4 · Telegram Chat ID

1 Kalshi account & API key

  • Create a Kalshi account (use the FlipPulse link) and set a password.
  • Verify your email (click the link Kalshi sends).
  • Complete identity verification if prompted (name, DOB, last 4 SSN).
  • Go to Settings → API Keys → Create API Key; name it FlipPulse.
  • Copy the API Key ID → write it as #1 above.
  • Download the PEM file (⚠ downloads once!) → save it; tick #2 above.

2 Install Telegram

  • Install the official Telegram (blue paper-plane, "Telegram FZ-LLC") on phone or computer.
  • Sign in with your phone number and the 5-digit code; add your first name.

3 Create your Telegram bot

  • Search BotFather; open the one with the blue ✓.
  • Tap Start, then send /newbot.
  • Give a name (e.g. FlipPulse Alerts) and a username ending in "bot".
  • Copy the token BotFather gives you → write it as #3 above. (Keep it secret!)

4 Start the bot & get your Chat ID

  • Open your new bot (the link from BotFather) and tap START. (Required!)
  • Search @userinfobot, open it, tap Start, send hi.
  • Copy the number after Id: → write it as #4 above.

✓ Finish

  • Open the Customer Onboarding Form, paste in #1, #3, #4, and upload the PEM (#2).
  • Submit & complete checkout. Done — your bot starts in paper mode automatically.
⚠ PEM downloads once. If you lose it, just create a new API key and use the new pair.
ℹ Money stays yours. The API key can only trade on your Kalshi account — it can't withdraw funds.
