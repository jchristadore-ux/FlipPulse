# 08 — Daily Operating Procedure

Print this. It is the whole job. Total weekday time: **~75 min/day** + Sunday
batch (~2.5 hrs). If a day collapses, do the ☆ items only (25 min) — the
streak matters more than the volume.

---

## MORNING — 8:00–8:35am ET

- [ ] ☆ **Metricool app check (2 min):** Planning → today. Any red/failed
      posts? Fix per `06_SCHEDULER_SETUP.md` §8.
- [ ] ☆ **Telegram sweep (5 min):** open operator chat → forward the 1–3 most
      interesting overnight bot alerts to t.me/flippulse → screenshot the best
      one into `/Content/WeekNN/Shots/` (this is today's `[shot]` material and
      tonight's post fuel).
- [ ] **Story (3 min):** 9am Metricool ping → post today's Story from the
      week's folder, add the interactive sticker (calendar says which ST-#).
- [ ] ☆ **X reply sprint (15 min):** open the "Engage" list → reply to 8–15
      posts using the §9 comment formula (substance from the doctrine, never
      links). Prioritize posts <30 min old.
- [ ] **Our-post replies (5 min):** answer every overnight comment/DM
      (templates: `03_POST_LIBRARY.md` §9).
- [ ] **News scan (5 min):** BTC moving hard? Kalshi/prediction markets in the
      news? → trigger the trend override (calendar standing rule 1): draft
      T46/T47/T48 with the 1.2 prompt, queue for the pm slot.

## MIDDAY — 12:45–1:00pm

- [ ] ☆ **Comment sweep (10 min):** all platforms, reply to everything new;
      pin the best comment on today's TikTok.
- [ ] **Save fuel (5 min):** any good question from comments → paste into the
      week's content sheet as a future T59 / F6 video prompt.

## EVENING — 8:00–8:30pm

- [ ] ☆ **Engagement round 2 (10 min):** 5 more X replies; 10 TikTok/IG
      comments on niche-adjacent rising videos.
- [ ] **Community (10 min):** Mon/Wed/Fri → one genuine contribution in FB
      groups; Tue/Thu → Reddit comments (r/Kalshi, r/algotrading); Kalshi
      Discord presence.
- [ ] ☆ **Log the day (5 min):** KPI sheet daily row: follower counts (quick
      glance per app), link clicks (dub.co), signups (Stripe/sheet auto),
      anything viral-ish. 5 minutes, no analysis — analysis is Friday's job.
- [ ] **Tomorrow glance (2 min):** calendar row for tomorrow — is its `[shot]`
      slot covered by this morning's screenshot? If not, grab one now.

---

## WEEKLY

### Sunday — Batch day (~2.5 hrs, the engine room)
- [ ] Film the week's videos (calendar's BATCH note lists which; scripts in
      `04_VIDEO_SCRIPTS.md`) — one setup, all takes back-to-back (~60 min).
- [ ] Edit in CapCut, double-export each (TikTok + clean) (~40 min).
- [ ] Run the Sunday batch prompt (`07_AUTOMATION.md` §1.1), edit output,
      slot into the library (~20 min).
- [ ] Schedule the full week in Metricool (`06` §4) (~30 min).
- [ ] Stage `/Content/WeekNN/` folders incl. 7 Stories (~15 min).

### Friday — Decision day (30 min, 3pm)
- [ ] Make.com digest arrived (Scenario B) — read it.
- [ ] Fill the weekly KPI row (Metricool Analytics + dub.co + MailerLite +
      Stripe tab) — `09_METRICS_KPI_DASHBOARD.md` §3.
- [ ] Apply the kill/double rules (09 §5): pick 1 thing to double next week,
      1 to kill. Write both into next week's calendar row notes.
- [ ] Check Metricool best-times heatmap → nudge slot times if it disagrees.
- [ ] Phase 3+: review any running boosts against the §6 kill rules.

### Any weekday (15 min total across the week)
- [ ] Mon: send/refresh 5 influencer DMs (log in CRM `Influencers` tab).
- [ ] Wed: babysit the week's Reddit post all afternoon (calendar weeks 3+).
- [ ] Thu: day-14 customers → testimonial/referral ask went out (automation
      does it; you handle replies personally).

---

## MONTHLY (last Friday, 90 min)

- [ ] **Numbers vs targets:** KPI sheet monthly block vs the 30/60/90 ladder
      (09 §4). Miss by >30% on two metrics → run the diagnosis tree (09 §6).
- [ ] **Content audit:** top 5 / bottom 5 posts of the month → update the
      Evergreen Bank notes; retire formats that failed twice.
- [ ] **Customer email:** send "Your bot this month" (07 §4.4).
- [ ] **Money:** marketing spend total, CAC, MRR — 10-minute P&L sanity check
      against `business/FlipPulse_Business_Model.xlsx`.
- [ ] **Library refill:** one Claude session to draft next month's new assets
      (T61+, V31+) from the month's real questions/logs.
- [ ] **Phase gate:** check `01_MASTER_MARKETING_PLAN.md` §12 — did we hit
      the exit criteria? If yes, start the next phase's new motions.

## Failure modes & the rule for each
| Situation | Rule |
|---|---|
| Missed a whole day | Do NOT double-post tomorrow. Resume the calendar as-is. |
| A post is bombing | Leave it. Never delete for performance (only for compliance). |
| A post is going viral | Clear 2 hours: reply to every comment; pin the CTA comment; same-day follow-up post referencing it. |
| Nasty comment thread | One calm factual reply (§9 template), then mute. Never argue twice. |
| BTC crashes / vol spike | Trend override: V29 + T47 + FB14. Empathy first, never gloat, disclaimer on everything. |
| A customer complains publicly | Reply publicly within 1 hour: acknowledge + take it to DM + resolve + (with permission) post the resolution. |
| You're burned out | Ship the ☆ items only. A 25-minute day beats a broken streak. |
